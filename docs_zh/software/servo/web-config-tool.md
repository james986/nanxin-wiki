## 1. 工具说明

该工具为浏览器版上位机，适用于 UART/RS485 总线舵机的基础调试与快速联调。  
当前已改为本地部署版本，入口页面位于本仓库：
`docs_zh/software/servo/data/web-controller/index.html`

## 2. 使用前准备

1. 确认舵机与转接板已正确接线，并接通舵机电源；
2. 使用 USB 数据线将转接板连接到电脑；
3. 建议使用最新版 Chrome 或 Edge 浏览器；
4. 首次连接串口时，请在浏览器弹窗中允许串口访问权限。

> [!WARNING]
> 网页调试工具依赖浏览器串口能力（Web Serial API），移动端浏览器通常不支持完整功能。

## 3. 舵机连接与识别

1. 打开网页调试工具，点击连接串口；
2. 在弹窗中选择对应串口设备并确认；
3. 点击扫描舵机，等待列表返回在线设备；
4. 选中目标舵机，查看当前状态和基础参数。

## 4. 常用调试功能

- 角度控制：发送目标角度并观察舵机响应；
- 状态回读：读取当前位置、运行状态等信息；
- 参数配置：读取或写入基础参数并即时生效；
- 多设备联调：扫描并切换不同 ID 舵机进行测试。

## 5. 常见问题

- 串口不可见：检查驱动安装、数据线与 USB 接口是否正常；
- 扫描无设备：检查舵机供电、电压范围与接线顺序；
- 连接失败：关闭其他占用串口的软件后重试；
- 写入无效：确认当前设备 ID 正确，且参数在允许范围内。
