## 2. UART 串口舵机



- **通讯方式**：UART 串口



UART 通讯一般使用两条数据线：**TX（发送）** 和 **RX（接收）**。在设备间连接时，一个设备的 TX 要接到另一个设备的 RX，反之亦然。



华馨京舵机在应用时，会通过 **UC-01 转接板**将标准的 UART **全双工（TX 与 RX 分开）**转换为 **半双工**方式。也就是说，TX 和 RX 信号被合并成一条单线 **S 信号**，实现收发共用线路，从而简化布线。

## 3. RS485 总线舵机



- **通讯方式**：RS485



RS485 采用的是**差分信号传输**，通过两根信号线 A、B 来共同传递数据。接收端根据 A、B 两线之间的电压差来判断逻辑状态：



– 当**A 电压 > B 电压** 时，表示逻辑 “1” （高电平）；



– 当B 电压 > A 电压 时，表示逻辑 “1” （高电平）；



这种方式的**优点**是：



1. 1. **抗干扰能力强**：在长距离传输时，信号容易衰减并受到噪声影响。但由于外部干扰通常会同时作用在 A、B 两根线（即共模干扰），接收端只关心两线之间的电压差，因此干扰会被抵消。

   2. **传输距离远**：相比单端信号，差分信号衰减更小，RS485 通常可以实现上百米甚至上千米的稳定通信。

   3. **共模抑制能力强**：接收器能够有效忽略 A、B 两线上的相同电压变化（即共模电压），只对电压差敏感，这种能力称为 **共模抑制**（CMRR）。



{{ fig_indent_adjustable("../images/image-20250929120434735.png","华馨京舵机通讯方式对比", "100%", "3.5em") }}



- **协议**：自研协议（和上述UART底层协议一致）



详细解读访问：[总线伺服舵机通信协议 – Fashion Star Wiki](https://wiki.fashionrobo.com/uartbasic/uart_rs485_protocols/)



- **功能**：单圈角度控制、多圈角度控制、阻尼模式、同步指令控制、异步指令控制、数据监控等，支持角度、温度、电压、电流、功率、状态反馈，可实现闭环控制。



- **应用场景**：工业设备、科研实验、医疗设备、需抗干扰性强、需远距离控制的系统。



## 4. CAN 总线舵机



- **通讯方式**：CAN（支持 CANBUS 与 UAVCAN）



- **功能**：高实时性，抗干扰能力强；可反馈角度、温度、电压、电流、功率、状态



- **应用场景**：无人机、自动驾驶平台、AGV 小车、需要高速可靠通讯的移动机器人



## 5. PWM 舵机



- **通讯方式**：标准 **PWM** 脉宽信号



舵机的控制信号为周期是**20ms 的脉宽调制**（PWM)信号，其中脉冲宽度从0.5ms-2.5ms，相对应舵盘的位置为0°-180°，给一个特定的PWM，舵机会转到**相应的角度**。



{{ fig_indent_adjustable("../images/image-20251009150742321.png","华馨京舵机通讯方式对比", "100%", "3.5em") }}



- **功能**：常见 PWM 控制，可通过华馨京PWM上位机修改保护参数，提供了全面的堵转、温度和电压保护，同时用户可以通过编程器对多项专业参数进行自定义设置。**不支持数据反馈**，**无法闭环**。



- **应用场景**：教育、航模、遥控车船、机械手臂初级应用、对成本敏感且无需反馈的场合。
