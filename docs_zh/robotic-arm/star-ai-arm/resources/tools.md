# 控制软件与工具
---

我们为 StarAI Arm 系列提供了配套的控制和调试工具。  
请通过下方链接跳转到对应页面，在 **软件与工具 > 机械手臂** 中访问和使用。

- **[PID调试工具 (Cello)](../../../software/robotic-arm/pid-tools.md)**

    用于 Cello 机械臂动力模式优化与参数写入，提供分场景调试流程和常见故障处理。

- **[桌面控制软件](../../../software/robotic-arm/web-config-tool.md)**

    用于机械臂连接、基础控制与参数配置，适合日常调试与联调。
