## 2. 有刷舵机和无刷舵机的精度与空载使用寿命对比

|                | 有刷舵机 |  有刷舵机  | 无刷舵机  |
| -------------- | :------: | :--------: | :-------: |
| **内部电机**   | 铁芯电机 | 空心杯电机 | 无刷电机  |
| **空载精度**   | 0.5°-1°  |  0.5°-1°   | 0.2°-0.3° |
| **空载使用寿命** |  100h  |    400h    |  2000h   |

用户可根据具体需求选择不同舵机。

其精度与空载使用寿命的区别，主要因为内部使用了不同的电机与不同齿轮材质。

## 3. 不同电机之间的区别

#### 3.1.1 铁芯电机

- 转子结构：转子由叠压的硅钢片铁芯构成，铁芯上有绕制的铜线圈。
- 换向方式：通过机械式的**碳刷**与转子轴上的换向器接触来切换电流方向，实现转子线圈的连续旋转。
- 优点：结构简单，制造成本低。
- 缺点：效率较低，寿命有限，有碳刷限制，噪音较大，散热较差，精度较低。

#### 3.1.2 空心杯电机

- 转子结构：转子没有铁芯，转子绕组被绕制成一个薄壁的杯状结构。
- 换向方式：通过机械式的**碳刷**与转子轴上的换向器接触来切换电流方向，实现转子线圈的连续旋转。
- 优点：结构简单，散热较好，能量密度高。
- 缺点：寿命有限，有碳刷限制，精度较低。

#### 3.1.3 无刷电机

- 定子/转子结构：定子通常是带有铁芯和绕组的（通常是三相绕组），转子是永磁体（通常是高性能钕铁硼磁钢）。
- 换向方式：完全摒弃了机械碳刷和换向器，通过**电机控制器算法**来检测转子位置（通常用霍尔传感器或无感算法），并精确控制给定子各相线圈通电的时序和电流大小/方向，产生旋转磁场“拉动”永磁转子转动。
- 优点：高效率，寿命长，无机械换向磨损件，低噪音，散热好：热量主要产生在定子绕，启动扭矩大。
- 缺点：控制复杂，成本高。
