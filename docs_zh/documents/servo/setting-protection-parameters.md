# 正确设置保护参数（功率、电压、温度）



### 1.1 电压保护



当总线伺服舵机实际工作电压超过或低于预设的保护阈值时，伺服舵机将自动卸力并进入保护状态。要使舵机恢复正常工作，需重新进行上电操作。



{{ fig_indent_adjustable("../images/image-2-1600x200.png","正确设置保护参数（功率、电压、温度）", "100%", "3.5em") }}



**注意：**由于部分电源上电速度较慢，可能触发低压保护。建议将低压保护阈值设定为4000(mV)，以减少误触发的可能性。设定过高的高压保护阈值，会造成内部元器件不可逆的损坏，建议不要随意修改出厂设定。



### 1.2 堵转失锁保护



此参数是设定总线伺服舵机在堵转情况下的应对策略。这个设定的效果为**堵转失锁保护开关**，开启之后舵机堵转后将失去锁力，进入自由释放状态。



{{ fig_indent_adjustable("../images/image-20240823143236416.png","正确设置保护参数（功率、电压、温度）", "100%", "3.5em") }}



### 1.3 功率保护



功率保护提供两种响应模式，需要结合**堵转保护开关**来达到相应的保护状态。



例如如果是夹爪类的应用，建议启用模式二，确保物件被判断夹持后，能用某个力度保持，同时不让伺服舵机过载；如果是以保护为目的的，建议选择模式一。



{{ fig_indent_adjustable("../images/image-7-1600x234.png","正确设置保护参数（功率、电压、温度）", "100%", "3.5em") }}



{{ fig_indent_adjustable("../images/image-11.png","正确设置保护参数（功率、电压、温度）", "100%", "3.5em") }}



- **模式一（堵转保护-开）**：当舵机功率达到预设的保护值时，舵机将自动卸力并进入保护状态。为恢复正常工作，需要重新上电复位。

- **模式二（堵转保护-关）**：当舵机功率达到设定值A（功率保护值）后，系统会自动调整功率，将其降至一个较低的预设值B（堵转功率上限），确保功率不超过B（注：B<A）。这有助于防止过载并确保设备在安全范围内运行。



### 1.4 温度保护



当舵机内部温度超过设定阈值时，舵机进入温度保护状态，防止舵机继续升温导致舵机损坏。



**注意：**设定过高的温度保护阈值，会造成硬件不可逆的损伤。
