# 连线方式及电源解决方案



## 1. 连线方式



总线伺服舵机根据分布位置和布线需求，可灵活采用串联、并联及串并结合的接线方式，实现了最简洁的布线设计。

## 2. 电源解决方案



考虑到单颗舵机的功率需求、长距离布线及数据线的承载能力，下面介绍了几种供电方案，这些方案不仅确保了舵机的功率供给，还优化了电源的分配。



### 2.1 常规走线方案



通过UC01转接板同时进行信号以及电源的转接



- 优点：走线简单，可以直接串接



- 缺点：每块转接板允许最大电流约20A，需要根据不同型号舵机的堵转电流进行规划，如超过数量则需要多块转接板。



比如单颗25kg舵机堵转最大电流为3A，则每块转接板，建议不能超过6颗舵机。



{{ fig_indent_adjustable("../../../images/2024082009480933-1600x900.png","连线方式及电源解决方案", "100%", "3.5em") }}



### 2.2 电源单独走线方案



  转接板只负责信号处理–半双工转全双工，只需要1块转接板即可；电源部分则直接接到舵机。



- **电源串联**



  串联的数量，取决于连线及接口的压降。



{{ fig_indent_adjustable("../../../images/2024082009524616-1600x900.png","连线方式及电源解决方案", "100%", "3.5em") }}



- **电源并联**



  如果舵机较多，考虑到线长后会存在压降，电源不进行串接，主电源单独走线并联到每一个舵机
