# 软件与工具

---
## 舵机系列

- **[PC 调试软件](./servo/pc-config-software.md)**

    提供ID修改、各种工作模式的实时控制、参数配置、状态监测与固件升级等功能，适用于本地调试。

- **[网页调试工具](./servo/web-config-tool.md)**

    基于浏览器在线调试，无需安装客户端，可进行基础参数设置、状态读取与通信测试。

- **[串口工具助手](./servo/serial-port-assistant.md)**

    使用串口工具助手，发送控制指令并回读数据，快速验证舵机通信与运行状态。


---
## 机械系列

- **[PID调试工具(Cello)](./robotic-arm/web-config-tool.md)**
    
    为 Cello 提供在线 PID 参数调试，支持多种机械臂运动特性的参数配置与快速适配。

- **[网页调试工具](./robotic-arm/web-config-tool.md)**
    
    基于浏览器提供机械手臂拖动控制与参数修改功能，便于在线调试与状态验证。



---
## 双足机器人

- **[动作编辑器](./humanoid/action-editor.md)**

    基于浏览器在线编辑机器人动作组，支持动作调试与打包导出，可用于直接遥控或对接其他平台调用。

- **[DB动作编辑器](./humanoid/fighter-editor.md)**

    面向格斗机器人场景的专用动作编辑器，基于 DBSP 舵机开发，支持动作组编辑、调试与下发，并可将动作映射至手柄按键，实现机器人实时操控。
