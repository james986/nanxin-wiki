# 多个舵机串联建议





## 1. 分组连线原因





多个舵机串联时普通开关电源的电流会不够，电流不够会影响舵机的正常运行，此时可以采用分组供电的方法。

## 4. 主要事项





- 将舵机分为几组，每组单独使用一个电源，每组**不超过6个舵机**。





- 组与组之间，组与转接板之间，只需连接**信号线(S)**与**负极(-)**,转接板再连接主控进行控制。
