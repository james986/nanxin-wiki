# 阻尼模式详解及应用





允许舵机在外部力的作用下调整到不同的角度位置，同时保持一定的阻尼效果，并可自定义阻尼系数。





多用于机械臂示教以及给舵机泄力。





{{ fig_indent_adjustable("../images/阻尼.png","阻尼", "100%", "3.5em") }}





如下视频，案例为模拟盖板应用。





盖板受重力影响自由下落，舵机在阻尼模式下，会对盖板的下落有阻尼效果。





范例演示不同的阻尼系数下，对盖板自由下落速度的影响。
