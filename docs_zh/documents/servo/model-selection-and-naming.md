# 产品选型 | 型号命名规则



## 1. 产品选型表



本选型表旨在为用户提供详细的产品选择参考，使用时请注意以下几点：



**协议兼容**：全系列产品的通信协议保持一致，不同型号可以混搭使用。



**结构兼容**：25-50kg系列产品在外观尺寸和安装孔位上保持统一，可无缝替换和升级；



**扭矩说明**：标识的扭矩参数是指静态扭矩，即产品在静止状态下所能承载的最大力量。在动态运动过程中，实际承载的扭矩通常为静态扭矩的30%-40%。在设计和应用中，请根据这一比例合理选择产品，以确保其能够在动态条件下安全、稳定地工作。



**电源选择：**请根据所选产品的电压和堵转电流，确保配置足够功率的电源，以避免因功率不足导致产品进入保护。



**特殊需求**：如果用户有特殊的功能需求，请联系技术支持获取进一步的咨询和定制服务。
