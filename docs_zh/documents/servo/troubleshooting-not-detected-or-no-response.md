# 舵机未检测 / 无法响应问题解析





## 1. 当上位机无法正常控制舵机检查步骤





### 1.1 检查供电电压





首先检查我们的供电电压是否在我们舵机规定范围内，可以查看舵机型号，型号**不带有H**的是低压版本（6.0v-8.4v），**带有H**的是高压版 本（9.0v-12.6v）。

{{ fig_indent_adjustable("../images/image-20250708105140427.png","舵机未检测 / 无法响应问题解析", "100%", "3.5em") }}





### 1.2 检查电源限流





检查电源是否有限流的情况，电源电流至少在3A以上，**请使用**稳定可靠、输出能力充足**的直流电源适配器或开关电源等供电设备供电。**避 免使用充电宝、干电池（包括电池组）或非专业的降压模块**，这些设备可能无法提供稳定电压和足够电流，或存在上电特性不匹配等问题。**





{{ fig_indent_adjustable("../images/wps1.jpg","舵机未检测 / 无法响应问题解析", "100%", "3.5em") }}





### 1.3 检查低压保护是否误触发





尝试调低低压保护阈值或更换电源，市面上部分电源上电速度较慢，舵机主控芯片工作电压为3.3v，当芯片上电完成（达到 3.3v）时，若检测到的电源输入电压仍低于设定的低压保护阈值，舵机会误触发低压保护。





{{ fig_indent_adjustable("../images/image-20250708095911284.png","舵机未检测 / 无法响应问题解析", "100%", "3.5em") }}





## 2. 当上位机无法扫描到舵机的解决方案





### 2.1 检查电源连接





请先检查**电源**、**转接板**和**总线伺服舵机**是否依次正确连接，确保接线无松动或错误，此时，转接板的电源指示灯应亮起。接线方式 如下图：





{{ fig_indent_adjustable("../images/2025070409274324.png","舵机未检测 / 无法响应问题解析", "100%", "3.5em") }}





（低压舵机外接电源范围6.0v-8.4v，高压舵机所需电压为9.0v-12.4v）





### 2.2 ID冲突问题





检查是否同时连接了多个舵机，因为出厂默认舵机的ID都是 **#0** ，如果没有修改ID就同时接入，就会出现无法扫描的情况。请先单独连接单 个伺服舵机，重启设备并扫描，扫描成功后，再依次为不同舵机分配ID编号，
